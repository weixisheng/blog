# Git emoji

An emoji guide for your commit messages

# emoji list

emoji 描述 图标
:art: 改进代码的结构/格式 🎨
:zap: 提升性能 ⚡️
:fire: 删除代码或文件 🔥
:bug: 修复问题 🐛
:ambulance: 关键修补程序 🚑
:sparkles: 新功能 ✨
:memo: 添加或更新文档 📝
:rocket: 部署 🚀
:lipstick: 添加或更新UI和样式文件 💄
:tada: 项目初始化 🎉
:white_check_mark: 添加或更新测试 ✅
:lock: 修复安全问题 🔒
:bookmark: 发布/版本标签 🔖
:rotating_light: 修复编译器/ Linter警告 🚨
:construction: 正在进行中 🚧
:green_heart: 修复CI构建 💚
:arrow_down: 降级依赖项 ⬇️
:arrow_up: 升级依赖项 ⬆️
:pushpin: 针对特定版本的依赖关系 📌
:construction_worker: 添加或更新CI构建系统 👷
:chart_with_upwards_trend: 添加或更新分析或跟踪代码 📈
:recycle: 重构代码 ♻️
:heavy_plus_sign: 添加依赖项 ➕
:heavy_minus_sign: 删除依赖项 ➖
:wrench: 添加或更新配置文件 🔧
:hammer: 添加或更新开发脚本 🔨
:globe_with_meridians: 国际化与本地化 🌐
:pencil2: 修复拼写错误 ✏️
:poop: 需要改进的错误代码 💩
:rewind: 恢复更改 ⏪
:twisted_rightwards_arrows: 合并分支 🔀
:package: 添加或更新已编译的文件或包 📦
:alien: 由于外部API更改，更新代码 👽
:truck: 移动或重命名资源（例如：文件，路径，路由） 🚚
:page_facing_up: 添加或更新许可证(license) 📄
:boom: 介绍 breaking changes 💥
:bento: 添加或更新assets 🍱
:wheelchair: 改善可访问性 ♿️
:bulb: 在源代码中添加或更新注释 💡
:beers: Write code drunkenly 🍻
:speech_balloon: 添加或更新文本和文字 💬
:card_file_box: 执行数据库相关的更改 🗃
:loud_sound: 添加或更新日志 🔊
:mute: 删除日志 🔇
:busts_in_silhouette: 添加或更新贡献者 👥
:children_crossing: 提高用户体验/可用性 🚸
:building_construction: 进行架构更改 🏗
:iphone: 响应设计工作 📱
:clown_face: mock数据 🤡
:egg: 添加或更新复活节彩蛋 🥚
:see_no_evil: 添加或更新.gitignore文件 🙈
:camera_flash: 添加或更新快照 📸
:alembic: 执行实验 ⚗️
:mag: 改善SEO 🔍
:label: 添加或更新类型 🏷
:seedling: 添加或更新种子文件 🌱
:triangular_flag_on_post: 添加，更新或删除功能标志 🚩
:goal_net: 捕获错误 🥅
:dizzy: 添加或更新animations transitions 💫
:wastebasket: 弃用需要清理的代码 🗑
:passport_control: 有关授权，角色和权限相关的代码 🛂
:adhesive_bandage: 简单修复非关键问题 🩹
:monocle_face: 数据探索/检查 🧐
:coffin: 删除dead code ⚰️
