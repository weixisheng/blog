# 脚本延迟

script标签中的async和defer

# defer

通知浏览器该脚本将在文档完成解析后，触发 DOMContentLoaded 事件前执行，相当于告诉浏览器立即下载，但延迟执行。

会按照顺序执行所有的script。

# async

对于普通脚本，如果存在 async 属性，那么普通脚本会被并行请求，并尽快解析和执行，不保证按照它们的先后顺序执行。
